# 🚀 Scallex

Plataforma moderna focada em conversão, performance e experiência para negócios digitais.

🔗 **Acesse o projeto:**  
👉 https://scallex.vercel.app/

---

## 🧠 Sobre o projeto

O **Scallex** é uma aplicação web desenvolvida com foco em:

- Alta conversão
- Performance otimizada
- Interface moderna e responsiva
- Experiência fluida para o usuário

A proposta é oferecer uma landing page estratégica capaz de apresentar soluções, capturar leads e direcionar usuários para ação.

---

## ⚙️ Tecnologias utilizadas

Este projeto foi construído utilizando:

- ⚛️ React + TypeScript  
- ⚡ Vite  
- 🎨 CSS moderno / responsivo  
- 📦 Componentização modular
